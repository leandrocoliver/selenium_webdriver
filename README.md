# Curso Selenium WebDriver - T:1606335
Repositório desenvolvido durante as aulas para demonstrar exemplos de testes automatizados com Selenium WebDriver

# Visão Geral
* Linguagem: Java
* Build/gestão de dependências: Maven
* Estrutura padrão de testes em src/test/java
* Exemplos de testes: com.test.WebElementsTest

# Requisitos
* Java 11+
* Maven 3.8+
* Navegador Brave (Google Chrome foi recomendado pelo curso)
* WebDriver gerenciado automaticamente via WebDriverManager
